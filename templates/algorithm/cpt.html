{% endblock %} {% block body_content %} Causal Path Tracing is an algorithm for causally tracing the internal pathways of a Transformer-based model's answer generation. The algorithm explores the causal path for each sample's model decision within a pretrained Transformer. It is evaluated based on how well the original decision is preserved when only the identified Causal Path remains.
- Hit Rate: The proportion of samples for which the original decision index is maintained when only the Causal Path is preserved. For instance, if among 100 explored samples, the original decision is maintained for only 70 samples when only the Causal Path is preserved, then Hit Rate = 0.7. - Faithfulness: The degree to which the logit value of the original decision index is preserved when only the Causal Path is preserved. For instance, if among 100 explored samples, the logit of the original decision index is maintained by only about 60% when only the Causal Path is preserved, then Faithfulness = 0.6.

* Interpretation Guidelines
Evaluation Result Interpretation
High Hit Rate & High Faithfulness The explored Causal Path is truly causal for that decision.
High Hit Rate & Low Faithfulness The explored Causal Path is causal enough to produce the decision, but there is some information loss.
Low Hit Rate & High Faithfulness Although the explored Causal Path maintains the original logit, it is difficult to consider it causal because the model's decision has changed.
Low Hit Rate & Low Faithfulness The explored Causal Path is not causal.
